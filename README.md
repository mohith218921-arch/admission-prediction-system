# Admission Prediction System

This project analyzes past admission data and predicts future admissions
for diploma departments.

## College
Nanjiah Lingammal Polytechnic College

## Features
- Year-wise admission analysis (2023–2025)
- Department-wise comparison
- Seat availability vs admission
- Prediction & suggestions

## Tools Used
- HTML, CSS
- PHP
- MySQL
- Power BI
- XAMPP
